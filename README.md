# Fandora 人工時管理系統

Fandora 內部工時追蹤與管銷費用分析 Dashboard。

## 功能

- **總表**：KPI 總覽、月份對比、前10專案、部門與員工分佈
- **專案分析**：各專案工時與管銷費用分攤
- **員工分析**：個人工時分佈、月趨勢、時薪成本
- **部門分析**：部門效率、成員貢獻、任務分佈
- **設定**：Google Drive 串接設定、資料管理

## 資料來源

HR 每月將工時日誌 Excel 上傳到 Google Drive 共享資料夾，Dashboard 自動讀取。

### 工時日誌格式

Excel 檔案需包含「明細總表」工作表，欄位：員工姓名、部門、日期、工作內容說明、實際時數

### 薪資資料格式

使用 salary-template.xlsx 範本，包含：員工姓名、部門、月份（YYYY-MM）、月薪

## 管銷費用計算

某人在某專案的成本 = (月薪 / 當月總工時) x 該專案工時

## 部署

本專案為純靜態網站，部署在 GitHub Pages。
